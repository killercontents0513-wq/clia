📋 시연 가이드 — 4분 코스

  1. SCENE 1 (1분) — 같은 내용, 5가지 얼굴.
    제품 진입 → 📝 MD Source → 우측 5개 탭 차례 클릭. "왼쪽 한 곳, 오른쪽 5개 채널"
  2. SCENE 2 (30초) — 한 줄 수정의 위력.
    가격 줄 수정 → 5개 출력 동시 갱신. "이게 5개 마켓 따로 수정하던 일"
  3. SCENE 3 (1분 30초) — Mercado Libre 에 실제 paste.
    📄 Plain Text 복사 → ML Vendedores 화면에 붙여넣기. "HTML 안 깨지고 구조 살아있음"
  4. SCENE 4 (1분) — 같은 원본, 다른 채널.
    🛒 Amazon Limited HTML → Amazon Seller Central 에 붙여넣기. 같은 원본이 채널별로 다르게 변환됨.
  5. SCENE 5 (1분) — 다국어 확장.
    Markdown 전체 복사 → ChatGPT/DeepL 에 "독일어로 번역, 구조 보존" → 결과를 다시 CLIA 에 paste → 즉시 독일어 5개 채널 생성
